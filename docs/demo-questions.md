# 시연 시나리오

면접/포트폴리오용 큐레이션 질문. 각 질문마다 *어떤 모드의 강점*이 드러나는지 명시 — 단일 모드만 보면 차이가 잘 보이지 않으므로 의도적 선택이다.

## KG 강점 — 인물/개념 관계 시각화 (4개)

| # | 질문 | 시연 포인트 |
|---|---|---|
| 1 | **예정론을 둘러싼 칼빈과 어거스틴의 관계는?** | Person↔Concept 양방향 관계, INFLUENCES/CITES 그래프 |
| 2 | 성례에 대해 칼빈이 누구를 인용하고 어떤 관점에 반대하는가? | Person+Concept+OPPOSES 시각화 임팩트 |
| 3 | 삼위일체에서 성부, 성자, 성령의 구별을 칼빈은 어떻게 정리했는가? | 3개 Concept 사이의 RELATED_TO/PART_OF 그래프 |
| 4 | 자유의지를 칼빈은 어떻게 정의하며, 어떤 관점과 대립하는가? | OPPOSES 관계 — "본성의 부패상", "교부들" 같은 대립 노드 |

## Hybrid 강점 — 토큰 스트리밍 + 정확한 본문 인용 (2개)

| # | 질문 | 시연 포인트 |
|---|---|---|
| 5 | 이신칭의의 정의를 한 단락으로 설명해줘 | 빠른 응답 + `st.write_stream` 토큰 단위 |
| 6 | 칼빈은 성례의 본질을 어떻게 설명했는가? 페이지 출처 인용 | source_pages expander, structured output `cited_pages` |

## Agentic 강점 — 도메인 외 거절 (1개)

| # | 질문 | 시연 포인트 |
|---|---|---|
| 7 | 오늘 날씨 어때? | 도구 호출 0회, "본 챗봇은 칼빈 신학에 한정된 답변만 가능합니다" |

## 보너스 — 3 모드 동시 비교 (1개)

| # | 질문 | 비교 포인트 |
|---|---|---|
| 8 | 칼빈은 예정론을 어떻게 정의하는가? | Hybrid(빠른 본문 발췌) vs Agentic(검색 자율 + 인용) vs KG(어거스틴/펠라기우스 관계 그래프 동시) |

## 시연 흐름 (5분 데모 가정)

```
0:00 ─ 챗봇 실행, 사이드바 모드 토글 보여주기
0:30 ─ Hybrid 모드 + Q5 (이신칭의 정의) — 토큰 스트리밍 시연
1:30 ─ Agentic 모드 + Q7 (날씨) — 도구 호출 0회, 도메인 외 거절
2:30 ─ KG 모드 + Q1 (예정론, 칼빈, 어거스틴) — 그래프 시각화 임팩트
3:30 ─ KG 모드 + Q2 (성례 비판 인물) — OPPOSES 관계 강조
4:30 ─ Q&A — "전체 1,251p 인덱싱 시 ~₩175" 등 비용 의식 어필
```

## 포트폴리오 어필 포인트 (면접관 청중)

1. **3 모드를 동일 인터페이스로 제공** — `PATTERN_NAME`, `query()`, `index_documents()` 통일 (RAG 패턴 학습 결과물)
2. **Hexagonal 분리** — KG 백엔드를 `KnowledgeGraphPort`로 추상화. Mock 어댑터로 단위 테스트 6/6 PASS, LLM/DB 호출 0회
3. **로컬/Aura 환경 추상화** — `.env`의 `NEO4J_URI` 한 줄로 환경 전환. 코드 무수정
4. **비용 의식 + 점진적 빌드** — 50샘플(₩17) → 단원별 균등 150청크(₩52 누적) → 시연. "전체 인덱싱 시 ~₩175" 명시
5. **단원 자동 추출 + 별도 에이전트로 검증** — 정규식으로 (책,장)→page 매핑 후 LLM 에이전트가 본문 sample 검증 → +1 보정
